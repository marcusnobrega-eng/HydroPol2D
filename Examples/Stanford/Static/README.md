# Stanford Static Rasters

Place the Stanford static rasters here before running the applied example.

Typical HydroPol2D static inputs include:
- `DEM.tif`
- `LULC.tif`
- `SOIL.tif`
- `LAI.tif`
- `Albedo.tif`
- `DTB.tif`
- `GW_table.tif`
- initial soil-moisture and initial water-depth rasters when required by the selected configuration

Do not place GPM rainfall rasters here for the spatially invariant Stanford example.
