# Stanford Applied Example

This folder is reserved for the Stanford HydroPol2D applied example. It is treated as an applied case study/example, not as the analytical validation benchmark for spatial rainfall.

Rainfall mode: spatially invariant rainfall. Unused GPM raster rainfall files have been removed from `Forcing/Rainfall` so the example does not imply satellite-raster forcing.

Before running the example, provide or verify the required static rasters under `Static/` and the required tabular input sheets under `Input_Data_Sheets/`. The current folder contains configuration and evapotranspiration inputs, but it is not yet a complete runnable example package.
