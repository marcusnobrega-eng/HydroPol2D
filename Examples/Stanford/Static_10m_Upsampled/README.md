# Stanford 10 m Upsampled Diagnostic Static Inputs

This folder contains a diagnostic 10 m static raster set generated by 3x
upsampling the reconstructed 30 m Stanford example rasters in
`../Static`.

These rasters are intended only for checking current HydroPol2D event
behavior at a finer numerical grid. They are not original 10 m Stanford
source data and should not be presented as a high-resolution observational
input product.

Resampling choices:
- `DEM.tif`: bilinear
- `LULC.tif`, `SOIL.tif`, `DTB.tif`, `Initial_Soil_Moisture.tif`,
  `LAI.tif`, `Albedo.tif`: nearest neighbor
