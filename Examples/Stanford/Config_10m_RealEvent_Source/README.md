# Stanford 10 m Real-Event Source Config

This folder preserves the downloaded Stanford 10 m bypass configuration for a
future real-event rerun.

The immediate website rerun uses the controlled design event
(`100 mm/h` for `1 hour`, simulated for `24 hours`) through the current
`run_stanford_local_case.m` runner. The files in this folder should be adapted
in a later step to reproduce the downloaded real-event setup with the current
HydroPol2D model.
